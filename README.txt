Integration for Merit Aktiva invoices.

== Description ==

* Automatically sends order to Merit Aktiva when order status is changed to "Completed".
* Provides new order action to generate Merit Aktiva invoice manually.

== Installation ==

1. Install plugin.
2. Add Merit Aktiva configuration in Woocommerce integrations section.
